UIDAI Analytics Hub

This repository is a Streamlit app (app.py). Vercel is primarily designed for static sites and serverless functions; it does not run a persistent Streamlit server, so deployments without a static entrypoint often show Vercel’s NOT_FOUND page.

Run locally

Install deps and start Streamlit: pip install -r requirements.txt then streamlit run app.py

Deploy options (recommended)

Use Streamlit Community Cloud, Render, Railway, Fly.io, or any Docker-capable host.

If you *intended* this to be a Vercel web app, you’ll want to migrate the UI to a Vercel-native framework (e.g. Next.js) and keep Python analytics behind APIs.
